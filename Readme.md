# Node JS Web Page

### Description
> **In this example we are developing a web page with node js and using ejs (template engine) to render the views in that format**

## Technologies employed

### Back-End

	1) NodeJS
    2) Express

### Front-End

	1) Html5
	2) Css3
	3) Bootstrap
	4) Ejs
	5) AnimateCss
	6) GoogleFonts
